## 🚀 Deploy to Vercel in 5 Steps

### Step 1 – Create a GitHub Repository
1. Go to **github.com** → click **"New repository"**
2. Name it: `boyz-party-tracker`
3. Set to **Private** (recommended)
4. Click **Create repository**

---

### Step 2 – Upload these files to GitHub
1. On the new repo page, click **"uploading an existing file"**
2. Upload ALL files/folders from this zip, keeping the folder structure:
   ```
   booze-party/
   ├── index.html
   ├── package.json
   ├── vite.config.js
   ├── public/
   │   ├── manifest.json
   │   ├── icon-192.png
   │   └── icon-512.png
   └── src/
       ├── main.jsx
       └── App.jsx
   ```
3. Click **Commit changes**

---

### Step 3 – Connect to Vercel
1. Go to **vercel.com** → Sign in with GitHub
2. Click **"Add New Project"**
3. Find your `booze-party-tracker` repo → click **Import**
4. Vercel auto-detects Vite. Just click **Deploy** — no changes needed!
5. Wait ~1 minute for deployment ✅

---

### Step 4 – Get your link
- Vercel gives you a link like: `https://boyz-party-tracker.vercel.app`
- Share this link with yourself via WhatsApp/Notes

---

### Step 5 – Install on iPhone (Add to Home Screen)
1. Open the link in **Safari** on your iPhone
2. Tap the **Share button** (box with arrow at bottom)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **Add**
5. App icon appears on your home screen — works like a native app! 🎉
